## Create Next.js project
  - To create a standard project, use the initial Next.js command
  
  ```bash
    npx create-next-app@latest . \
    --typescript \
    --eslint \
    --tailwind \
    --app \
    --no-src-dir \
    --turbopack \
    --import-alias "@/*" \
    --use-pnpm \
    --git \
    --force
  ```
